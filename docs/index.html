Monokai-Earth-Forest

Dark theme for people who wish they were in a forest.

Supports js, html, css, python and php.

Nature is great, go outside!

Can't? Well at least you've got this nature inspired theme to help you cope.

Monokai-Earth-Forest is only available for Visual Studio Code. It comes with predefined syntax highlighting colors. Uses earth-tones to be easy on the eyes, for who like greens, browns, and yellows. An IntelliJ version is in the works.

Designed and developed by Sam Lazrak.
